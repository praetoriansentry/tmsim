Instructions for using the TM Simulator:
  1. Load your instruction set. Each instruction should look like: {A,B}->{C,D,E}
    A
    The current state
    B
    The current symbol on the tape
    C
    The new state to move to
    D
    The symbol to write on the tape
    E
    The Direction to move the tape either. Must be R or L
    Every state, symbol, or direction should be just one character. Use the state H for the halting state. Use an underscore _ for a blank symbol on the tape.
  2. Set the intial symobls on tape and move the tape to the correct position.
  3. Click the current state at the top of the page and set it to the state you want to start with.
  4. Press start. If the touring machine halts, it will stop with state H.
If you'd like to see the simulator in action try running the demo.
